Introduction

  • The World Wide Web as we know it is based on HyperLinks
  • Links are the defining feature of the web because they allow you to move from one web page to another — enabling the very idea of browsing or surfing
  • A hyperlink is a location in a document that links to another location in that document or links to another document or resource

Understanding URLs

  • http://www.mywebsite.com/categories/shoes.html
  • http:// is the communication protocol
  • www.mywebsite.com is the domain of my website
  • categories is a room/folder in my website
  • shoes.html is the HTML document

Writing Links

Writing links

  • Links are created using the <a> element
  • Users can click on anything between the opening <a> tag and the closing </a> tag
  • You specify which page you want to link to using the href attribute.

Using absolute hyperlink URLs

  • Absolute URL’s are most of the time used for external links
  • Example:
    • <a href=“http://www.microsoft.com”>Microsoft’s Website</a>

Using relative hyperlink URLs

  • Relative URL’s are most of the time used for pointing to an internal resources
  • Example:
    • <a href=“/categories/shoes.html”>Shoes</a>

Directory Structure

Directory Structure

On larger websites it's a good idea to organize your code by placing the pages for different section of the site into a separate folders

Folders on a website are sometimes referred to as directories

The top-level folder is known as the root folder

Adding Target

  • the target attribute specifies how you want to open the HyperLink
  • Target can be:
  • Example:
    • _blank - Opens the linked document in a new window or tab
    • _self - Opens the linked document in the same frame as it was clicked (this is default)

Anchors

  • In HTML, when saying anchor element it is referred to pointing out to a specific part of the same page using id="ID" attribute (which can be used on every HTML element).
  • To link to an element that uses an id attribute you use the <a> element again, but the value of the href attribute starts with the # symbol, followed by the value of the id attribute of the element you want to link to.

Anchors

	<a href="#arc_shot">Arc Shot</a><br />
	<a href="#interlude">Interlude</a><br />
	<a href="#prologue">Prologue</a><br /><br />
<h2 id="arc_shot">Arc Shot</h2>
	<p>A shot in which the subject is photographed by an

<h2 id="interlude">Interlude</h2>
	<p>A brief, intervening film scene or sequence, not
	specifically tied to the plot, that appears

<h2 id="prologue">Prologue</h2>
	<p>A speech, preface, introduction, or brief scene
	preceding the the main action or plot of a film;
	contrast to epilogue</p>
	<p><a href="#top">Top</a></p>

Link to e-mail addresses

  • Linking to email addresses is very similar as other simple links / hyperlinks
  • You only use mailto:<email-address> inside the href of the element
  • Exapmle:
    • <a href="mailto:jon@example.org">Email Jon</a>
