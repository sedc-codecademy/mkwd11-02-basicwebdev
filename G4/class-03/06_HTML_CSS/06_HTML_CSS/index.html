Introduction

Introduction

  • Tables are used for displaying data in a proper horizontal (rows) and vertical (columns) order
  • A table represents information in a grid format.
  • Think of them as an Excel tables or spreadsheets
  • In HTML, tables can be achieved in a simple way using dedicated table tag with nested table headers, rows, data and footer tags

Don't use tables to create layouts on HTML pages.

HTML Table Elements

  • <table> - Specifies a table
  • <caption> - Specifies a table caption
  • <thead> - Specifies table header row
  • <tbody> - Specifies table body rows
  • <tfoot> - Specifies table footer rows
  • <tr> - Specifies a table row
  • <th> - Specifies a table header cell
  • <td> - Specifies table data cell

HTML Table Elements Attributes

  • <table border="n"> - Specifies table border thickness in pixels
  • <table cellpadding="n"> - Specifies padding in table cell
  • <table cellspacing="n"> - Specifies cellspacingbetween cells

More on tables here: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/table

Basic Table Structure

Basic Table Structure


<table border="1">
	<tbody>
		<tr>
			<td>Row 1, Column 1</td>
			<td>Row 1, Column 2</td>
		</tr>
		<tr>
			<td>Row 2, Column 1</td>
			<td>Row 2, Column 2</td>
		</tr>
	</tbody>
</table>

Complete Table Structure


<table border="1">
	<thead>
		<th>Header Row 1, Column 1</th>
		<th>Header Row 1, Column 2</th>
	</thead>
	<tbody>
		<tr>
			<td>Row 1, Column 1</td>
			<td>Row 1, Column 2</td>
		</tr>
		<tr>
			<td>Row 2, Column 1</td>
			<td>Row 2, Column 2</td>
		</tr>
	</tbody>
	<tfoot>
		<th>Footer Row 1, Column 1</th>
		<th>Footer Row 1, Column 2</th>
	</tfoot>
</table>

Spanning Columns

  • colspan
  • Sometimes you may need the entries in a table to stretch across more than one column
  • The colspan attribute can be used on a <th> or <td> element and indicates how many columns that cell should run across

Spanning Rows

  • rowspan
  • You may also need entries in a table to stretch down across more than one row
  • The rowspan attribute can be used on a <th> or <td> element to indicate how many rows a cell should span down the table
