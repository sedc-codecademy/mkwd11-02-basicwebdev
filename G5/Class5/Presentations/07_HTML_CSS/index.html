Introduction

Introduction

  • Traditionally, the term 'form' has referred to a printed document that contains spaces for you to fill in information
  • HTML borrows the concept of a form to refer to different elements that allow you to collect information from users of your site

The best known form on the web is probably the search box that sits right in the middle of Google's homepage

Form Structure

Form Structure

  • <form>
  • The <form> element should always carry the action and method attributes
  • The action attribute defines the action to be performed when the form is submitted
  • The method attribute specifies the HTTP method (GET or POST) to be used when submitting the form data

More on forms here: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/form

Basic Form Structure


<form action="subscribe.php" method="GET">
	This is where the form controls will appear...
</form>

Drop Down / Select List Box

Drop Down / Select List Box

HTML5 form validation

  • required attribute
  • Validation helps ensure the user enters information in a form that the server will be able to understand when the form is submitted.
  • *required is a boolean attribute

HTML5 specific elements

  • <input type="date" />
  • <input type="email" />
  • <input type="url" />
  • <input type="search" value="Search" />

Placeholder

  • placeholder attribute
  • On any text input, you can also use an attribute called placeholder whose value is text that will be shown in the text box until the user clicks in that area
  • Don't use placeholder as a raplacement for label
